# `clawdfun setup`

Initialize `~/.clawdfun/clawdfun.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
clawdfun setup
clawdfun setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
clawdfun setup --wizard
```
