# `clawdfun pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
clawdfun pairing list whatsapp
clawdfun pairing approve whatsapp <code> --notify
```
